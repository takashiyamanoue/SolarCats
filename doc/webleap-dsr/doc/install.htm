インストール

       WebLEAP は Java のプログラムです。 WebLEAP をダウンロードするコンピュータに、JDK 1.3 以上の java SDK または、JRE がインストールされていることを確認してください。これは、コマンドプロンプト(DOS 窓)で、java –version コマンドを実行することによって確認することができます。

       ここをクリックして、dsr.jar　を、WebLEAPをインストールするディレクトリにダウンロードしてください。dsr は、WebLEAP を含んでいる、分散システムツールです。

       dsr.jar をダウンロードしたディレクトリで、コマンドプロンプトを起動します。

       「jar  xvf  dsr.jar　」 を実行することによって、dsr.jar を展開します。Dsr という名のディレクトリが作成されます。

       これでインストールの終了です。
